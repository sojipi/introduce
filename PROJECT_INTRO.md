# EdgeShowcase - 基于 ESA 全栈生态的动态个人展示平台

## 项目简介
**EdgeShowcase** 是一个基于阿里云 ESA（边缘安全加速）全栈生态构建的动态个人作品展示与管理平台。它突破了传统静态博客/简历更新繁琐的局限，利用 ESA 的边缘计算能力，实现了一套无服务器（Serverless）、数据动态管理、全球极速访问的个人名片系统。

---

## 💡 创意卓越：让简历“活”在边缘
**“不仅仅是静态页面，而是一个完整的边缘原生应用。”**

*   **动态交互体验**：不同于传统的 Hexo/Jekyll 等静态站点，本项目提供了一个功能完备的 **Admin 管理后台**。用户可以直接在浏览器中登录后台，对个人技能、项目经历、获奖情况进行增删改查（CRUD），数据实时更新，无需重新构建部署。
*   **数据可视化仪表盘**：后台集成了数据可视化 Dashboard，实时展示技能分布、项目统计等关键指标，界面美观，交互流畅，让个人实力的展示更加直观和专业。
*   **极简架构美学**：摒弃了传统的“前端+后端服务器+数据库”的臃肿架构，完全依托边缘节点能力，实现了“代码即基础设施”的轻量化设计理念。

---

## 🚀 应用价值：部署即用的开发者利器
**“为开发者量身打造，解决个人展示站点的维护痛点。”**

*   **解决核心痛点**：传统的 GitHub Pages 等静态托管方案，每次修改内容都需要修改代码、Git 提交、等待构建流水线。EdgeShowcase 允许用户随时随地通过管理后台更新内容，修改立即生效（得益于 KV 存储），极大降低了维护成本。
*   **开箱即用（Deploy-Ready）**：项目具备极高的通用性。任何开发者、设计师或求职者只需将项目一键部署到阿里云 ESA，即可拥有自己的动态官网。无需购买云服务器（ECS），无需配置 Nginx，无需维护数据库，真正做到了**低成本、零运维**。
*   **极速访问体验**：得益于 ESA 全球边缘节点的加速能力，无论访问者身处何地，都能享受到毫秒级的页面加载速度，极大提升了个人品牌的触达效率。

---

## 🛠️ 技术探索：深度挖掘 ESA 边缘全栈能力
**“100% Edge-Native，无源站架构的完整实践。”**

本项目深入探索并应用了 ESA 的四大核心边缘能力，展示了边缘计算在全栈开发中的巨大潜力：

1.  **ESA Pages (静态托管)**
    *   托管 Vue.js 构建的 SPA（单页应用）前端资源。
    *   利用 Pages 的构建能力，自动化部署前端代码。

2.  **Edge Routine (边缘函数)**
    *   **智能路由**：接管所有请求流量，实现 SPA 的路由重写（Rewrite）和 404 回退（Fallback）逻辑，确保单页应用在边缘节点的正常运行。
    *   **API 转发**：将 `/api/*` 请求无缝转发至云函数处理，实现了前后端的同构部署。

3.  **ESA KV Storage (边缘存储)**
    *   **替代传统数据库**：项目完全摒弃了 MySQL/MongoDB，使用 **Edge KV** 存储所有动态数据（用户信息、项目列表、技能数据等）。
    *   **高可用与低延迟**：利用 KV 的全球同步特性，实现了数据的就近读取，让数据库查询速度媲美静态资源加载。

4.  **ESA Cache (边缘缓存)**
    *   **精细化缓存策略**：通过配置，对带 Hash 指纹的静态资源实行强缓存（1年），对 HTML 入口实行协商缓存，对 API 接口实行不缓存。
    *   **性能极致优化**：配合忽略查询字符串等高级配置，最大化缓存命中率，显著降低了回源次数，节省成本的同时提升了用户体验。

---

**总结**：EdgeShowcase 不仅仅是一个个人网站，它是一个**基于边缘计算架构的现代化 Web 应用范本**，完美诠释了“Serverless + Edge”如何赋能开发者构建更轻、更具、更快的应用。
